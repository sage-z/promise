# Promise 承诺

一个接口设计管理、调试测试、定义分发工具

![You know,](./app/renderer/view/welcome/youknow.jpg)

愿在数据对接的路上，多些真诚，少些套路

## 特性
* Based on the OpenAPI v3.0
* 可离线使用
* 分布式
* 版本管理，分发
* 支持mock
* 跨平台
* 接口测试
* 接口调试
* 多节点（代理）
* sdk多端发布
* 多协议
* 版本管理，分发
* 无文档
* 国际化
